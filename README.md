# Сайт одностраничник "Научиться учиться"

Информационный проект, который собрал в себя перечень современных и эффективных подходов к обучению. На сайте представлена иформация по техникам обучения, размещенные с промощью grid, видео с использованием API Youtube и ссылка на книгу и дополнительные ресурсы для самостоятельного обучения.

Планы по доработке:
*подключить другие шрифты;
*добавить любимые образовательные видео;
*проверить код на кроссбраузерность и дописать все вендорные префиксы;
*задизайнить форму, через которую пользователи смогут отправить вам комментарий.
